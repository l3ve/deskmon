# Deskmon

Deskmon 是一只生活在 macOS 桌面上的像素小宠物，也是一组克制的本地效率工具。

它会在你划定的区域里活动、休息和追逐鼠标，并通过同一只宠物承载倒计时、截图、文本取用和本地自动化提醒。所有核心数据留在本机，不依赖云服务。

> 当前版本为 `0.1.0`，仅面向 macOS。项目仍在开发阶段，暂未提供面向普通用户的正式安装包。

## 当前能力

### 桌面小宠物

- 使用透明无边框窗口显示淡紫白像素宠物。
- 支持拖拽、隐藏、暂停移动、置顶和一键移回活动区域。
- 可设置小、中、大三种尺寸，以及安静、标准、活泼三种活动节奏。
- 可限定活动区域，宠物会在可见工作区和自定义区域内活动。
- 鼠标在附近快速移动时，宠物会注意并短暂追逐；鼠标靠近准备操作时会主动停下。
- 右键宠物打开轻量原生菜单，左键单击不会触发操作。

### 单次倒计时

- 设置一个 `1–180` 分钟的默认时长。
- 从菜单栏或宠物右键菜单直接开始倒计时。
- 运行中可查看剩余时间或安静取消。
- 自然结束后立即回到未运行状态，并发送“XX 分钟倒计时结束啦”。
- 不包含专注/休息阶段、暂停、自动重复、历史统计或任务管理。

### 统一宠物提醒

宠物可见时，倒计时完成、截图结果、“记住刚想到的”和本地 CLI 消息都会进入宠物旁的像素对话框。

消息严格按顺序显示，支持重复计数和最多四层叠卡。对话框出现时宠物保持原位置和常态尺寸；拖拽、悬停或截图期间会暂停或延迟展示。

宠物隐藏时，内部消息改用 macOS 系统通知。重新显示宠物后不会补播旧消息。

### 记忆力

“记忆力”窗口管理三类本地文本：

- **记忆中**：运行期间自动捕获的纯文本剪贴板历史，最多 10 条，退出后清空。
- **笔记本**：用户主动保存的文本，最多 50 条，加密保存在本机。
- **变量**：手动维护的 `key/value/note`，最多 50 条，适合 token、邮箱、地址和命令参数。

变量的 value 默认隐藏，不参与列表预览或搜索。复制变量只会写入剪贴板，不会自动粘贴，也不会进入“记忆中”；可选择在 30 秒后仅当内容仍匹配时自动清空剪贴板。

Deskmon 不是密码管理器。“笔记本”和“变量”的本地加密用于避免普通明文落盘，不提供抵抗本机攻击者的强安全承诺。

### 区域截图与标注

- 从菜单栏或宠物右键菜单进入覆盖层框选。
- 在原位置使用矩形、箭头和文字标注，并支持撤销。
- 只有点击保存才会生成最终 PNG；丢弃或在框选阶段按 `Escape` 不会留下文件。
- 默认保存到桌面，也可在设置中选择其他目录。
- 截图不会写入剪贴板、记忆力或截图历史。
- 需要 macOS 屏幕录制权限；Deskmon 使用稳定签名身份保持开发构建间的授权连续性。

### 本地提醒 CLI

在设置的“命令行工具”面板中，可以安装或更新 `/usr/local/bin/deskmon`：

```bash
deskmon notify --title "Codex" --text "任务执行完成"
```

`--text` 必填，`--title` 可选。Deskmon 未运行时命令会静默成功，不会自动启动应用；该接口仅用于当前 macOS 用户的本地调用。

## 使用入口

菜单栏提供完整控制：显示或隐藏宠物、暂停或恢复移动、移回活动区域、开始或取消倒计时、区域截图、记忆力、快速保存当前剪贴板、设置和退出。

宠物右键菜单保持轻量，只提供倒计时、区域截图、回忆、暂停或恢复移动及隐藏宠物。内容管理、CLI 安装和活动区域配置都放在独立窗口中。

## 本地开发

需要 Node.js、npm、Rust 和可用的 macOS Tauri 开发环境。

安装依赖并启动开发版：

```bash
npm install
npm run tauri dev
```

运行前端检查和测试：

```bash
npm run build
npm test
```

运行 Rust 检查和测试：

```bash
cargo check --manifest-path src-tauri/Cargo.toml
cargo test --manifest-path src-tauri/Cargo.toml
```

打包 macOS 应用：

```bash
npm run tauri -- build
```

项目配置使用 `Apple Development` 签名。执行打包后，应按项目规则覆盖安装并重新打开应用，再完成真实桌面验收。
