# Deskmon

Deskmon 是一个 macOS 桌面小宠物。它会待在桌面上活动、休息、陪你计时，也能帮你轻量保存和找回一些常用文本。

它的目标是提供一个安静、克制、不打扰的桌面陪伴体验。

## 主要功能

### 桌面小宠物

- 透明小窗口，常驻在桌面安全区域内。
- 默认角色是和应用 icon 一致的淡紫白像素小宠物。
- 会自己移动、停留、休息，也可以被拖拽。
- 支持隐藏、暂停、移回活动区域。
- 右键宠物可以打开小菜单。
- 可以调整大小、置顶状态和活动区域。
- 可以选择安静、标准、活泼三种活动节奏。

### 计时器

- 支持 1 / 5 / 10 / 25 分钟计时。
- 同一时间只运行一个计时器。
- 计时中可以查看剩余时间或取消。
- 计时完成后会发送 macOS 通知。
- 宠物可见时，会短暂变大并播放庆祝动画。

### 记忆力

Deskmon 可以帮你保存三类文本：

- 记忆中：运行期间自动记录的纯文本剪贴板历史，退出后清空。
- 笔记本：你主动保存的文本，会加密保存在本机。
- 变量：你手动维护的 key/value 内容，适合 token、邮箱、地址、命令参数等常用文本。

变量的 value 默认隐藏。点击变量时，只会把 value 写入剪贴板，不会自动粘贴到当前应用。

## 使用方式

### 菜单栏

从 macOS 菜单栏图标可以：

- 显示或隐藏宠物
- 暂停或恢复宠物活动
- 启动计时器
- 打开记忆力
- 打开设置
- 移回活动区域
- 退出应用

### 宠物右键菜单

右键宠物可以快速：

- 启动计时器
- 查看或取消当前计时
- 回忆最近保存的文本
- 隐藏宠物
- 打开设置

### 记忆力窗口

在记忆力窗口里可以：

- 查看临时剪贴板历史
- 把内容保存到笔记本
- 管理笔记本内容
- 新增、编辑、删除变量
- 搜索记忆中、笔记本和变量
- 复制内容回剪贴板

## 隐私说明

“记忆中”只保存在运行期间，退出应用后会清空。

“笔记本”和“变量”会加密保存在本机。Deskmon 的目标是避免常用文本明文落盘，但它不是完整密码管理器。

Deskmon 不会自动粘贴内容，也不会把变量 value 展示在列表、菜单或搜索结果里。

## 开发运行

安装依赖：

```bash
npm install
```

开发运行：

```bash
npm run tauri dev
```

构建检查：

```bash
npm run build
```

打包应用：

```bash
npm run tauri -- build
```
