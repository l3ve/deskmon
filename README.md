# Deskmon

Deskmon 是一个 macOS 桌面小宠物。它会待在桌面上活动、休息、陪你计时，也能帮你轻量保存和找回一些常用文本。

它的目标是提供一个安静、克制、不打扰的桌面陪伴体验。

## 主要功能

### 桌面小宠物

- 透明小窗口，常驻在桌面安全区域内。
- 默认角色是和应用 icon 一致的淡紫白像素小宠物。
- 会自己移动、停留、休息，也可以被拖拽。
- 在宠物附近快速移动鼠标时，它会先注意并短暂追逐；鼠标靠近准备操作时会停下，不影响拖拽和右键菜单。
- 支持隐藏、暂停、移回活动区域。
- 右键宠物可以打开小菜单。
- 可以调整大小、置顶状态和活动区域。
- 可以选择安静、标准、活泼三种活动节奏。

### 专注计时

- 默认支持 5 / 25 / 45 分钟专注，并在专注完成后衔接 5 分钟休息。
- 可以在设置里修改专注时长、休息时长和完成提示文案。
- 专注中可以从菜单查看剩余时间或取消本轮。
- 专注结束后，宠物会放大并停在当前屏幕中央，让你选择休息、再专注 5 分钟或结束本轮。
- 选择休息后，宠物会保持在屏幕中央并显示休息倒计时。
- 休息结束后，可以继续原专注时长、再休息 5 分钟或结束本轮。
- 每一步都由你确认，Deskmon 不会自动开始休息或下一轮专注。
- 专注结束和休息自然结束后都会发送 macOS 通知。
- 休息结束时可以播放系统通知声音。

### 记忆力

Deskmon 可以帮你保存三类文本：

- 记忆中：运行期间自动记录的纯文本剪贴板历史，退出后清空。
- 笔记本：你主动保存的文本，会加密保存在本机。
- 变量：你手动维护的 key/value 内容，适合 token、邮箱、地址、命令参数等常用文本。

变量的 value 默认隐藏。点击变量时，只会把 value 写入剪贴板，不会自动粘贴到当前应用。

### 区域截图

- 从菜单栏或宠物右键菜单选择“区域截图”，即可进入 macOS 原生矩形框选。
- 截图固定保存为 PNG，默认放在桌面，也可以在设置中选择其他文件夹。
- 截图成功后会发送系统通知；按 `Escape` 取消时不会产生文件或提示。
- 截图期间宠物、窗口和专注计时保持原样，框选到 Deskmon 时它也会出现在图片中。

## 使用方式

### 菜单栏

从 macOS 菜单栏图标可以：

- 显示或隐藏宠物
- 暂停或恢复宠物活动
- 启动专注计时
- 开始区域截图
- 打开记忆力
- 打开设置
- 移回活动区域
- 退出应用

### 宠物右键菜单

右键宠物可以快速：

- 启动专注计时
- 查看或取消当前计时
- 开始区域截图
- 回忆最近保存的文本
- 隐藏宠物
- 打开设置

### 记忆力窗口

在记忆力窗口里可以：

- 查看临时剪贴板历史
- 把内容保存到笔记本
- 管理笔记本内容
- 新增、编辑、删除变量
- 搜索记忆中、笔记本和变量
- 复制内容回剪贴板

## 隐私说明

“记忆中”只保存在运行期间，退出应用后会清空。

“笔记本”和“变量”会加密保存在本机。Deskmon 的目标是避免常用文本明文落盘，但它不是完整密码管理器。

Deskmon 不会自动粘贴内容，也不会把变量 value 展示在列表、菜单或搜索结果里。

## 开发运行

安装依赖：

```bash
npm install
```

开发运行：

```bash
npm run tauri dev
```

构建检查：

```bash
npm run build
```

打包应用：

```bash
npm run tauri -- build
```
