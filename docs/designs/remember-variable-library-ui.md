## 信息架构

“记忆力”窗口保持一个工作台结构：

- 顶部：标题、数量摘要、来源筛选、搜索、新增变量。
- 左侧：回忆列表。可以看全部，也可以只看记忆中、笔记本、变量。
- 右侧：当前选择的详情、变量表单或操作反馈。
- 宠物小菜单：只做“回忆”取出，不做新增、编辑、删除或搜索。

## 状态

1. 默认工作台
   - 默认可以停在“全部”来源，也可以记住用户上次选择的来源。
   - 左侧按来源分组展示条目；变量条目只显示 key 和 note，不显示 value。
   - 右侧展示当前选中条目。变量详情默认隐藏 value。

2. 新增变量
   - 点击“新增变量”后，右侧详情区切换为表单。
   - 表单字段为 key、value、note。
   - 保存成功后回到该变量详情，并选中新变量。
   - 取消后回到进入表单前的选择。

3. 表单校验
   - key 为空、value 为空、key 重复等错误直接显示在字段下方。
   - 校验错误不使用系统弹窗。

4. 编辑变量
   - 点击“编辑”后，右侧详情区切换为编辑表单。
   - value 可以临时显示；切换条目、取消、保存、关闭窗口后恢复隐藏。
   - 保存成功后回到详情态。

5. 搜索
   - 搜索框覆盖记忆中、笔记本、变量。
   - 变量只匹配 key 和 note，不匹配 value。
   - 搜索结果继续按来源分组。

6. 复制反馈
   - 点击“复制 value”后，只显示 key 级反馈。
   - 如果自动清理开启，右侧状态条显示剩余清理时间。
   - 如果剪贴板内容变化，清理任务取消，状态条可以显示“已取消清理”。

7. 删除确认
   - 删除变量使用系统确认。
   - 确认文案只显示 key，不显示 value。
   - 确认或取消后，焦点回到“记忆力”窗口。

8. 宠物小菜单回忆
   - “回忆”下分记忆中、笔记本、变量。
   - 变量子菜单只显示 key。
   - 点击变量后把 value 写入剪贴板，不自动粘贴。

## 视觉原则

- 主界面只展示当前任务所需的信息，不把所有说明性规则铺在画面里。
- 变量 value 默认隐藏，视觉上把 key 作为主识别信息。
- 搜索、新增、复制、编辑、删除都放在用户自然寻找的位置；高风险操作使用红色但不扩大面积。
- 自动清理是设置和状态，不抢占主流程。
- 宠物小菜单保持原生轻入口，不承载管理界面。
