# Deskmon V1 Smoke Checklist

用途：每次准备交付可试用版本前，在真实 macOS 桌面确认当前主流程没有退化。

## 准备

- 关闭旧 Deskmon 进程。
- 打包应用，覆盖安装并重新打开。
- 确认应用签名有效。
- 确认 macOS 已授予所需的通知和屏幕录制权限。
- 如需覆盖显示器变化，准备连接或断开一次外接显示器。

## 宠物窗口

- 启动后能看到透明无边框的淡紫白像素宠物。
- 宠物位于可见工作区内，不被 Dock 或菜单栏遮住。
- 左键单击没有操作；左键拖拽不会跳到鼠标中心。
- 松手后宠物停在释放位置附近，并能继续自主活动。
- 悬停时自主移动暂停，动画与倒计时继续。
- 鼠标离开后能恢复自主移动或休息，不会永久卡在 idle。
- 小、中、大尺寸切换后，宠物和窗口边界都正确。
- 安静、标准、活泼三档在移动频率和速度上有可感知差异。
- 默认置顶开关能即时改变宠物窗口层级。

## 鼠标追逐

- 在宠物外圈快速移动鼠标时，宠物会先注意再短暂追逐。
- 普通缓慢经过不会频繁误触发。
- 鼠标进入点击安全内圈后，追逐立即取消。
- 准备拖拽或右键时，宠物不会继续逃离鼠标。
- 单次追逐不会持续跟随，不会越过活动区域。
- 暂停移动、倒计时、拖拽、截图和宠物提醒期间不会触发追逐。
- 追逐结束后经过观察和冷却，宠物能恢复普通活动。

## 菜单栏

- 菜单栏图标能打开 macOS 原生菜单。
- “隐藏宠物”和“显示宠物”能正确切换可见性。
- “暂停移动”和“恢复移动”能正确切换自主移动。
- “移回活动区域”会显示宠物、归位并保存位置。
- 未运行时显示“XX 分钟倒计时”。
- 运行中显示剩余时间和“取消倒计时”，菜单刷新不会自动关闭。
- “区域截图”“记忆力”“记住刚想到的”“设置”和“退出”都能执行。

## 宠物右键菜单

- 右键宠物打开原生小菜单，左键不打开菜单。
- 未运行时显示“XX 分钟倒计时”。
- 运行中显示剩余时间和“取消倒计时”。
- 提供区域截图、回忆、暂停或恢复移动和隐藏宠物。
- 不包含内容管理、CLI 安装、活动区域设置或退出入口。

## 单次倒计时

- 设置页可保存 `1–180` 分钟的单个默认时长。
- 非法值显示错误且不保存。
- 从菜单栏和右键菜单启动的是同一套倒计时。
- 启动后宠物进入 `timer-waiting`，本轮分钟数保持锁定。
- 运行中修改默认分钟数只影响下一轮。
- “取消倒计时”立即回到未运行状态，不发送完成消息。
- 自然完成后立即回到未运行，并生成“XX 分钟倒计时结束啦”。
- 系统休眠或阻塞后按真实结束时间结算，不依赖 tick 次数。
- 退出应用后不会恢复未完成的倒计时。

## 统一宠物提醒

- 宠物可见时，CLI、截图、快速保存剪贴板和倒计时完成使用宠物对话框。
- 对话框出现时宠物保持原位置和常态尺寸，不放大或移动到屏幕中央。
- 普通和成功消息显示 4 秒，错误消息显示 8 秒。
- 正文使用逐字显示；点击当前卡片后立即显示完整正文。
- 多条不同消息严格按到达顺序显示，当前消息不被打断。
- 重复消息合并计数，视觉叠卡最多显示四层。
- 连续发送超过 20 条时，当前消息保留，待播队列按既定淘汰规则工作。
- 屏幕右侧空间不足时对话框切到左侧；左右都不足时放到下方。
- 对话框和宠物整体拖动后保持新锚点，松手后重新选择布局。
- 悬停、拖拽或截图期间，消息暂停或延迟，状态结束后继续。
- 对话框临时置顶但不抢键盘焦点。
- 宠物隐藏时改发 macOS 通知，重新显示后不补播旧消息。

## 记忆力

- 启动时当前剪贴板只作为基线，不进入“记忆中”。
- 新复制的纯文本进入“记忆中”，去重并限制在 10 条。
- “记忆中”只在运行期存在，重启后清空。
- “记住它”和“记住刚想到的”把文本保存到加密笔记本。
- 笔记本限制在 50 条，支持多条置顶和取消置顶排序。
- 忘记笔记本内容时显示原生确认，确认和取消后窗口都恢复焦点。
- “回忆”只写回剪贴板，不自动粘贴。
- 变量支持新增、编辑、删除、复制和临时显示 value。
- 变量列表、搜索、菜单、确认和反馈不暴露 value。
- 变量搜索只匹配 key 和 note。
- 复制变量不会进入“记忆中”。
- 开启自动清理后，30 秒到期仅在剪贴板仍匹配该 value 时清空。

## 区域截图与标注

- 菜单栏和右键菜单进入同一套覆盖层框选。
- 覆盖层按显示器建立；选区不能跨越鼠标按下时所在显示器。
- 小于 `10 × 10 px` 的范围不进入编辑状态，可以继续框选。
- 有效框选后冻结选区，选区外保持灰色遮罩。
- 工具栏优先位于选区下方，与选区保持 8px 间距；空间不足时移到上方。
- 矩形、箭头和文字标注在缩放显示器上位置正确。
- 撤销按创建顺序移除上一项标注。
- 框选阶段按 `Escape` 或点击丢弃都会清理临时文件，不生成最终 PNG。
- 只有点击保存才在目标目录生成最终 PNG。
- 文件名使用 `Deskmon_YYYY-MM-DD_HH-mm-ss.png`，重名时追加序号且不覆盖。
- 最终 PNG 包含底层桌面和标注，不包含覆盖层、遮罩、工具栏或鼠标。
- 默认保存到桌面，自定义目录可保存并在重启后恢复。
- 保存失败时编辑内容保留；修复目录后需要再次点击保存。
- 缺少屏幕录制权限时显示说明，并只在用户确认后打开系统设置。
- 截图不会写入剪贴板、记忆力或截图历史。
- 截图进行中再次触发不会建立第二个会话。

## 设置与 CLI

- 设置窗口在默认尺寸下可滚动查看全部面板，不裁切活动区域预览。
- 活动区域预览随窗口尺寸使用真实可见宽高重绘。
- 自定义活动区域太小时不保存，合法范围保存后限制宠物移动。
- “重置默认”清除自定义活动区域。
- 截图目录支持选择文件夹和恢复桌面默认目录。
- “命令行工具”正确显示未安装、已安装、可更新或冲突状态。
- 安装或更新后 `/usr/local/bin/deskmon` 指向当前 Deskmon CLI。
- 同名非 Deskmon 命令不会被覆盖或删除。
- 卸载后只移除 Deskmon 自己安装的命令。
- `deskmon --version` 和 `deskmon --help` 输出正确。
- `deskmon notify --text "测试"` 能显示提醒。
- `--title` 可选，缺少 `--text` 或使用未知参数时返回非零状态。
- Deskmon 未运行时 `notify` 静默成功，且不会自动启动应用。

## 恢复与长时间运行

- 重启应用后恢复宠物可见性、暂停状态、尺寸、活动档位、置顶和合法位置。
- 显示器变化导致旧位置不可见时，宠物回到可见工作区。
- 菜单栏“移回活动区域”能兜底找回宠物。
- 连续运行 30 分钟后，宠物仍能活动、休息、追逐和响应菜单。
- 空闲时 CPU 没有持续异常升高，内存没有持续增长。
- 长时间运行后，倒计时、提醒、截图和记忆力仍可正常使用。
